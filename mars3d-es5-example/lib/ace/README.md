介绍：
  Ace是一个用JavaScript编写的独立代码编辑器。我们的目标是创建一个基于浏览器的编辑器，以匹配和扩展现有本机编辑器（如TextMate，Vim或Eclipse）的功能，可用性和性能。它可以很容易地嵌入到任何网页或JavaScript应用程序中。Ace是Cloud9 IDE的主编，也是Mozilla Skywriter（Bespin）项目的继任者。
